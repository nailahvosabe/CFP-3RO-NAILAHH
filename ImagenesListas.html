Los vinculos o los links son accesos a otras paginas o recursos den la web.
tienen un atributo propio que se llama href, que define la ubicacion a donde se dirije el vinculo
la etiqueta <> tiene apertura y cierre. en el contenido es lo que se ve en la pagina

Lo que se ve

el atributo href puede adoptar tres formas distintas:

  • Vinculos Relativos:como las imagenes, tienen una ruta usando como referencia el lugar donde se encuentre el documento .html
  • Vinculo Absoluto: esto tambien es lo mismo que con las imagenes. Podemos usar una url completa. Siempre teniendo en cuenta no usar un link de busqueda sino la url concreta del recurso a vincular
  • Vinculo Ancla:los vinculos ancla se usan para referenciar a las distintas partes del mismo documento. Para utilizarlo se debe usar el atributo global id que debe ser unico en el documento

Por otro lado los vinculos tienen otro atributo llamado target

Toca bigotudo

El atributo target junto con rel definen como se va a comportar el vinculo, en este caso usando el valor _blank nos abre una nueva pestaña y sin referencias a la pagina actual
